Architecture
How It Works

A four-step pipeline that transforms speech into polished, ready-to-use text — entirely on-device.

Step 1
Record
Hold hotkey and speak. VAD detects silence, auto-stops after 3s.
→
Step 2
Transcribe
faster-whisper (large-v3-turbo) on GPU converts speech to raw text.
→
Step 3
AI Cleanup
Local LLM (Qwen2.5-7B) fixes grammar, punctuation, tone & formatting.
→
Step 4
Paste
Clean text is pasted directly into the active window via clipboard.
Entire pipeline runs on your NVIDIA RTX GPU — zero network calls
Capabilities
Key Features

Everything runs locally. No cloud dependency for any feature.

100% Local Processing
No audio or text ever leaves your machine. CMMC 2.0, ITAR, and HIPAA compliant by architecture, not policy.
AI Text Cleanup
A 7B-parameter LLM corrects grammar, adds punctuation, adjusts tone, and formats text — no more editing raw transcripts.
6 Dictation Presets
General, Email/Formal, Chat/Casual, Code, Technical/Docs, and Custom. Each adapts the LLM's output style.
Push-to-Talk
Hold Right Ctrl to record, release to process. VAD auto-stops after 3s silence. Hold or toggle modes available.
Personal Dictionary
Add industry jargon, names, and acronyms. Whisper biases recognition toward your custom vocabulary.
Snippet Library
Trigger phrases that expand to full text blocks. Say your trigger, get your boilerplate inserted automatically.

Technical
Under the Hood

Built for NVIDIA RTX workstations that organizations already own.

AI Models
  • Whisper large-v3-turbo via faster-whisper
  • Qwen2.5-7B Q4_K_M GGUF via llama-cpp-python
  • Both models run on GPU with CUDA acceleration
  • ~4GB VRAM for Whisper + ~5GB for LLM
Requirements
  • NVIDIA RTX GPU with 8GB+ VRAM
  • Windows 11
  • CUDA 12.x toolkit
  • Python 3.12+
  • 5-minute install, auto-downloads models
Architecture
  • Python desktop app with pywebview UI
  • System tray with color-coded status
  • WebRTC VAD for silence detection
  • Windows registry auto-start support
  • Config stored at %APPDATA%
