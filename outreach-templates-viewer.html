r/LocalLLaMA
I built a local voice-to-text app that uses Whisper + a local LLM to clean up dictation — no cloud, everything on your GPU

Hey r/LocalLLaMA,

I have been building a Windows dictation app called Flow Dictation and wanted to share it here since this community is basically the reason it exists. The core idea: push-to-talk voice dictation where transcription AND text cleanup both run locally on your NVIDIA GPU. No cloud APIs, no data transmission, no subscriptions.

How it works:

  • Hold hotkey (Right Ctrl by default) and speak
  • Release hotkey — audio gets sent to faster-whisper (large-v3-turbo) running on your GPU
  • Whisper output goes to a local LLM (Qwen2.5-7B, Q4_K_M GGUF via llama-cpp-python with CUDA)
  • LLM cleans up the text — punctuation, grammar, formatting, sentence structure
  • Cleaned text gets pasted into whatever app you are working in

End-to-end latency is about 2-3 seconds for a typical dictation. The LLM cleanup step is what makes this genuinely usable as a daily driver — raw Whisper output is fine, but the LLM turns it into text you would actually send in an email or put in a document.

Dictation presets: General, Email/Formal, Chat/Casual, Code, Technical/Docs, and Custom (write your own system prompt). Other features include VAD with silence timeout, snippet library, personal dictionary, system tray with color-coded status, and a settings GUI.

Hardware: NVIDIA RTX with 8GB+ VRAM (tested on RTX 5070 Ti 16GB). ~4GB for Whisper + ~5GB for the LLM. Windows 11, CUDA 12.x, Python 3.12+.

Feedback wanted: Linux support demand? Would a 3B cleanup model be good enough for 8GB cards? Live transcription preview? Dictation history? Model swapping?

This community taught me most of what I know about local inference, so it felt right to share it here first.
